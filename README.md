# hamingja
hamingja is loyalty points system using non-tradable SLP token.  SLP transactions are combined with BCH checkdatasig covenants.

## Setting up test environment


### hamingja
```
yarn install
```
```
ionic serveKey component
```
### hamingja-admin
```
yarn install
```
```
ionic serve
```
### hamingja-server
```
yarn install
```
```
yarn dev
```
