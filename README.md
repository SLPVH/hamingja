# hamingja
hamingja is loyalty points system using non-tradable SLP token. SLP transactions using BCH checkdatasig covenants enabled this functionality.

## Setting up test environment


### hamingja
```
yarn install
```
```
ionic serveKey component
```
### hamingja-admin
```
yarn install
```
```
ionic serve
```
### hamingja-server
```
yarn install
```
```
yarn dev
```
